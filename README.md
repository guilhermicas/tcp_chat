# how to use

### Server side

python tcp_chat.py --username [USERNAME] --host [LOCAL_IP] --port [PORT] --client 0 \[--debug [0/1]]

### Client side

python tcp_chat.py --username [USERNAME] --host [REMOTE_IP] --port [PORT] --client 1 \[--debug [0/1]]
